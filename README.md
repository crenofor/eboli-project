Como executar o projeto
1. Pré-requisitos

Antes de começar, você precisa ter instalado:

Node.js (versão 16+ recomendada)
https://nodejs.org

NPM

Você não precisa instalar nenhum banco de dados externo, porque o projeto usa SQLite, que funciona em arquivo local.

2. Clonar o projeto ou baixar

git clone https://github.com/seu-repositorio/eboli-store.git
cd eboli-store

Ou apenas extraia o zip e entre na pasta do projeto.

3. Instalar as dependências

No terminal, execute:
npm install


Isso instalará:
express
express-session
bcryptjs
sqlite3
ejs
